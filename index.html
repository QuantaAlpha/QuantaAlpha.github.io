RepoMaster · Sema Code

Autonomous Exploration

RepoMaster (NeurIPS 2025 Spotlight) autonomously searches, analyzes, and explores large GitHub repositories—using hierarchical code trees, module-dependency and function-call views, and pruned context to complete complex tasks with far lower token use. Sema Code decouples the agent engine as embeddable infrastructure: the same core powers VS Code and messaging-style clients with multi-tenant isolation, scheduling, and layered permissions.
